## 功能和目的

- [X] 安全地加密需要保护的内容
  - [X] 使用尽可能强的加密，同时足够灵活便于配置算法参数
  - [X] 任意文件加密
- [X] 安全地使用密码
  - [X] 自动锁定
- [X] 实用功能
  - [X] 随机生成
  - [X] OTP

## 依赖

- Python 3.13或更高版本
- UPX
  - 仅构建时使用

## 快速开始

> `pip install -r requirements.txt`
>
> `python gui_main.py`

## 构建

### Windows

> `./build-gui.bat`
